# internship-projects

project Selections
Project 1: Smart Frame Gate (Option A) – An implementation that filters and processes frames based on specific quality gates.

Project 2: Scan Mode Stabilizer (Option B) – A tool designed to reduce jitter and stabilize input during active scanning modes.



How to Run the Projects
Both projects follow a standard IDE-based workflow:

Download: Clone this repository or extract the ZIP file.

Open IDE: Launch your IDE (e.g., Android Studio or VS Code).

Select Project: Navigate to the specific folder for either Project A or Project B.

Execute: Build the project and run it directly through the IDE onto a connected device.
